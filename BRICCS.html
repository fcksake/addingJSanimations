A novel physical to digital interface system that aims to teach young students about website design and programming.










in collaboration with Lucas Moiseyev, Claire Chen, Arka Roy
tools Java
Arduino (uno)
CMUcam5 Pixy(board optimized for computer vision)
physical prototyping
skills project management, hardware/software interfacing, design research, UX


Coded in Java, bri{css} is a toolkit intended to educate and introduce children to coding concepts in a more tangible way. This project aims to answer the question: how does adding a physical interface impact technology education? As the team came together, we were all, on some level, trying to code our own websites.




WHAT IS?

Bri{css} is a toolkit that combines physical understanding with a digital interface to help visually introduce basic web development.




HOW?

The design features physical toy blocks representing different elements of a website as well as a computer vision camera and texture localization matrix to determine the block locations. The software interface will take information about the block types and their locations to generate viable HTML, CSS, and Javascript code.





MY ROLE

As project manager, I was responsible for the proof of concept, hardware, and the physical toolkit. In addition, I collaborated on the visuals for the graphic user interface design.
