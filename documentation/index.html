Getting Started

Puppin is a HTML/CSS template based on Bootstrap and Bootstrap framework. It allows to built impressive, better & faster, mobile friendly websites.

HTML Structure

This is the base of each HTML file:

                <!-- Start of nav bar 
                ============================================= -->
                <nav id="puppin-nav">
                ...
                </nav>
                <!-- End of nav bar 
                ============================================= -->


                <!-- Start of header
                ============================================= -->
                <header id="site-header">
                ...
                </header>
                <!-- End of header
                ============================================= -->

                <!-- Start of section name
                ============================================= -->
                <section id="section-name-id" class="section-name-class">
                ...
                </section>
                <!-- End section name
                ============================================= -->

                
                <!-- Start of footer
                ============================================= -->
                <footer>
                ...
                </footer>
                <!-- End of footer
                ============================================= -->
